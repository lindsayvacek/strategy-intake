# Strategy Intake

A new-client intake form for The Design Kollective.

This is a static HTML/CSS/JS project hosted on GitHub Pages and embedded
on the live DK site at https://thedesignkollective.com.

## Stack

- Static HTML
- React + Babel standalone (loaded from unpkg)
- Vanilla CSS

## Local preview

Open `index.html` in a browser. No build step.
